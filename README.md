# Twinx AI Portal

Next.js (App Router) + TypeScript + Tailwind CSS.

## Első lépések

```bash
npm install
npm run dev
```

Nyisd meg a [http://localhost:3000](http://localhost:3000) címet a böngészőben.

## Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS** (v4)
- **Supabase** (Auth, Database, Storage) — később kerül bekötésre
